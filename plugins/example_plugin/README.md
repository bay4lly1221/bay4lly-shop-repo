# Example Plugin

A sample plugin to demonstrate Bay4lly Shop functionality.

## Features

- Simple menu integration
- Notification system
- External link support

## Installation

Install through Bay4lly Shop Developer Portal.

## Usage

1. Open Blockbench
2. The plugin will be automatically loaded
3. Look for "Bay4lly Example" in the menu

## Support

For issues or questions, visit the Bay4lly Shop repository.
